# Дневник на расположение (Mood Tracker)

Едноставна веб-апликација за следење на дневното расположение, со мотивациони пораки и визуелна "градина" која расте со секој запис.

Проектот е направен со чист **HTML, CSS и JavaScript** - без framework-и и без backend.

## Демо

Отвори го `index.html` во кој било модерен browser (Chrome, Firefox, Edge). Не е потребна инсталација, сервер или интернет конекција за да работи (освен за вчитување на фонтовите при прв пристап).

## Функционалности

- **Избор на расположение** - пет опции, од "Одлично" до "Ужасно"
- **Мотивациони пораки** - при секој избор се прикажува случајна порака поврзана со тоа расположение
- **Кратка белешка** - опционално поле за запишување што предизвикало одредено чувство
- **Историја на записи** - листа на сите претходни внесови со датум, расположение и белешка, со можност за бришење
- **Градина** - секој запис додава ново растение во визуелна лента, како симболичен приказ на изминатите денови
- **Статистика** - брoj на последователни денови (streak) и вкупен број записи

## Чување на податоци

Апликацијата користи `localStorage` во browser-от - записите остануваат зачувани и по затворање на табот или browser-от, без потреба од база на податоци или сервер.

Податоците се врзани за browser-от и патеката од каде е отворен фајлот. Бришење на browser историјата/cookies или отворање во incognito мод ќе ги избрише записите.

## Структура на проектот

```
mood-tracker/

```

## Технологии

- HTML5
- CSS3 (custom properties, flexbox)
- Vanilla JavaScript (ES6+)
- Web Storage API (`localStorage`)
- Google Fonts (Fraunces, Inter)

## Локално стартување

```bash
git clone https://github.com/<username>/mood-tracker.git
cd mood-tracker
```

Потоа само отвори го `index.html` во browser (двоклик или "Open with").
